Смотреть результат
  1. Выбрать, скачать, установить и запустить среду разработки.
  2. Создать «сайт» в этой среде.
  3. Создать веб-страницу (т.е. одну HTML страницу). Это будет страница вашего веб-портала.
  4. Приготовьте файл с картинкой для страницы вашего портала
  5. Прикрепите изображение к странице портала
  6. Создайте одну или две дополнительных страницы вашего сайта. Их содержание может быть любым и должно отражать ваши интересы.
  7. Создайте гиперссылки на страницы портала
  8. Создайте гиперссылку на сайт dvfu.ru (должен открываться всегда в новой вкладке браузера).
  9. Скопируйте ваши файлы страниц и картинку на web-сервер. Она должна быть доступна всем желающим в интернете. Рекомендуется в качестве хостинга использовать Github Pages, либо бесплатный web-хостинг
